# Turov_Akhmadiev

## Краулер

1. Файл **`index.txt`** формируется с помощью скрипта **`links_generator.py`**.  
   Скрипт собирает ссылки на статьи с сайта https://elementy.ru и сохраняет их в `index.txt`.

2. Краулер:
   - считывает ссылки из файла **`index.txt`**;
   - загружает содержимое каждой страницы;
   - сохраняет полученные файлы в директорию **`downloads`**.
