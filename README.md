# Turov_Akhmadiev

## Краулер

1. Файл **`index.txt`** формируется с помощью скрипта **`links_generator.py`**.  
   Скрипт собирает ссылки на статьи с сайта https://elementy.ru и сохраняет их в `index.txt`.

2. Краулер:
   - считывает ссылки из файла **`index.txt`**;
   - загружает содержимое каждой страницы;
   - сохраняет полученные файлы в директорию **`downloads`**.

## Обработка текстов и лемматизация

1. Скрипт:
   - считывает текстовые файлы из директории **`downloads`**;
   - очищает содержимое от HTML-разметки, спецсимволов и лишних пробелов;
   - приводит текст к нижнему регистру.

2. Обработка текста:
   - выполняется токенизация текста (разбиение на слова);
   - удаляются стоп-слова русского языка;
   - оставляются только корректные русские слова;
   - выполняется морфологический разбор слов;
   - отбрасываются слова с низкой вероятностью распознавания.

3. Результат:
   - формируются файлы **`...tokens.txt`** — список уникальных токенов;
   - формируются файлы **`...lemmas.txt`** — словарь лемм, где каждой нормальной форме соответствует список словоформ.
